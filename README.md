# 八皇后
規則
我每下一個皇后直線以及斜線上不得有其他皇后
EX : 

錯誤範例
|      | col1 | col2 | col3 | col4 |
| ---- | ---- | ---- | ---- | ---- |
| row1 |   Q  |   *  |   *  |   *  |
| row2 |   *  |   Q  |   *  |   *  |
| row3 |   *  |   *  |   *  |   Q  |
| row4 |      |      |      |      | 

正確範例
|      | col1 | col2 | col3 | col4 |
| ---- | ---- | ---- | ---- | ---- |
| row1 |   *  |   Q  |   *  |   *  |
| row2 |   *  |   *  |   *  |   Q  |
| row3 |   Q  |   *  |   *  |   *  |
| row4 |   *  |   *  |   Q  |   *  | 

以Q為中心算米字內都不得有Q(皇后)存在

```
*  *  *
 * * *
  *** 
***Q***
  *** 
 * * *
*  *  *
```

程式修改成N皇后後，
只要修改N即可運算出其他皇后結果。
```
int qNum = N;
```
